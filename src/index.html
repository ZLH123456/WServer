WELCOME TO THE WSERVER IMPLEMENTATION

To download a file just type the filename and extension in the url. Ex: index.html (this will access the page you're looking at). The following items are avaliable to download/visualize.

File Extension Description
fichero txt A file that contains some text.
index html An html page.If the ASCII parameter is set on true it will display this same page but without html tags, in other words, it will be displayed as a plain text.
image jpeg An image that will show in the browser. If the parameters are zip or gzip (or even both) you will get a windows to download the file compressed in the parameters that are set on true.

The parameters you're able to use are: asc (to display a html file as a plain text), zip and gzip (to download a file compressed on zip, gzip or both).
